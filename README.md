# Егор Летов — пространство

Сайт-трибьют Егору Летову и группе «Гражданская оборона». Шесть песен с текстами, обложками, плакатами, историей и живым блоком отзывов.

## Технологии

- Frontend: HTML/CSS/JS, nginx
- Backend: Node.js (Express)
- База данных: PostgreSQL
- Оркестрация: Docker Compose

## Запуск

### Требования

- Docker
- Docker Compose

**Важно:** проект работает только в Docker. Без контейнеров не запустится.

### Команды

```bash
# Клонировать репозиторий
git clone <ссылка>

# Перейти в папку
cd <название-папки>

# Запустить
docker-compose up -d
